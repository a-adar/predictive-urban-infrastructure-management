\# Predictive Urban Infrastructure Management



\## Overview

This project focuses on predictive traffic management in a simulated urban environment.



The system uses machine learning to forecast short-term congestion and dynamically adjust traffic signal timings to improve traffic flow.



\## Key Features

\- Traffic simulation with multiple intersections

\- Data collection from simulation

\- Time-series forecasting of congestion

\- Adaptive traffic signal control

\- Comparison with fixed-time baseline

\- Performance evaluation under normal and disrupted conditions



\## System Architecture

Simulation → Data Collection → Forecasting Model → Traffic Controller → Dashboard



\## Technologies

\- Python (pandas, numpy, scikit-learn)

\- SUMO (traffic simulation)

\- Streamlit (dashboard, later stage)
