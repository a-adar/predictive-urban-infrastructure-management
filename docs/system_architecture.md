\# System Architecture



\## Overview

The system consists of five main components:



1\. Traffic Simulation

&#x20;  - Generates vehicle flow and intersection states.



2\. Data Collection

&#x20;  - Logs traffic metrics such as queue length and waiting time.



3\. Forecasting Model

&#x20;  - Predicts short-term congestion using historical data.



4\. Traffic Controller

&#x20;  - Adjusts signal timings based on predicted congestion.



5\. Dashboard

&#x20;  - Displays system state, predictions, and performance metrics.



\## Data Flow

Simulation → Logs → Forecast Model → Controller → Evaluation/Dashboard
